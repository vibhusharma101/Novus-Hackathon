# EPRx Exchange Design System

## Brand
High-Contrast Corporate Minimalism. B2B fintech/trading aesthetic. "Calculated Trust."
Minimal shadows — use tonal layering + micro-borders for depth.

## Colors
- Primary Emerald: #059669 (buy actions, success, growth)
- Trust Indigo: #6366f1 (navigation, secondary focus)
- Background: #f8fafc (Slate-50)
- Cards: #ffffff with 1px Zinc-200 border
- Text primary: #0f172a (Slate-900)
- Risk Red: #ef4444
- Transition Amber: #f59e0b

## Typography
- Headlines: Geist (negative letter-spacing, tight)
- Body: Inter
- ALL numbers/data/₹/IDs/weights: JetBrains Mono (monospace, right-aligned in tables)

## Spacing
- 4px base grid
- 1440px max container
- Cards: 8px radius
- Buttons/inputs: 6px radius
- Table cells: 0px radius

## Components
- Buttons: Solid Emerald-600, white text, 6px radius, no shadow
- Tables: Slate-100 header, white rows, 1px bottom border, hover Slate-50
- Inputs: Zinc-200 border, 6px radius; weight/price fields use monospace
- CPCB Badge: Shield icon, Zinc-900 border, Emerald checkmark, "CPCB VERIFIED" data-sm
- Status pills: low-sat background, high-sat text
- Cards: 1px Zinc-200 border, no shadow, white bg, 8px radius
